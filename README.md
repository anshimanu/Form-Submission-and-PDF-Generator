# Form-Submission-and-PDF-Generator

## About

This project is a web application that allows users to submit a form and generate a PDF document based on the form data. The backend is built with Python FastAPI and uses LibreOffice in headless mode to convert filled Word document templates into PDF. The frontend is built with React and provides a user-friendly interface for form filling and submission.

## Features

- Dynamic form rendering with validation
- Real-time PDF generation from submitted data
- Downloadable PDF output
- Responsive and clean user interface
- Backend error handling and logging
